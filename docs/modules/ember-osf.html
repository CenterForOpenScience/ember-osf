This module is a rollup of the following modules:

  • adapters
    Base adapter class for all OSF APIv2 endpoints
  • authenticators
    Ember-simple-auth compatible authenticator based on session cookie. Intended to be used with the authorizer of the same name.
  • authorizers
    Ember-simple-auth compatible authorizer based on session cookie. Intended to be used with the authenticator of the same name.
  • components
    Lists citations for node in APA, MLA, and Chicago formats
  • const
  • helpers
    Generate a unique HTML element ID for this element. Given "someid" for component instance 123, returns "ember123-someid" Useful to ensure unique IDs, eg for when component is reused in page.
  • mixins
    Replacement for Ember-simple-auth AuthenticatedRouteMixin. Instead of redirecting to an internal route, this mixin redirects to CAS login URL, and brings the user back to the last requested page afterwards For OAuth this is done via the state parameter, and for cookies this is done via the service parameter. (TODO: Need a mixin that detects this!)
  • models
    Model for OSF APIv2 citation styles
  • serializers
    Base serializer class for all OSF APIv2 endpoints. Provides custom behaviors for embeds, relationships, and pagination.
  • services
    Access information about the currently logged in user
  • transforms
    Custom string field transform that uses the fix-special-char utility function to clean up malformed text sent from the server. This allows string fields to be correctly and transparently used in templates without manually fixing these characters for display on each use. This transform is used when fixstring is passed as the type parameter to the DS.attr function. `app/models/score.js import DS from 'ember-data'; export default DS.Model.extend({ astring: DS.attr('fixstring'), }); `
